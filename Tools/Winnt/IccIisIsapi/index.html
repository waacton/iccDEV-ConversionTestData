Interactive endpoints

These links hit the IIS-hosted native DLL directly. The upload console lives on endpoints.html.

./iccIisIsapi.dll?mode=health ./iccIisIsapi.dll ./iccIisIsapi.dll?format=xml ./endpoints.html

Integration posture

  • The IIS site root can stay locked down. Directory browsing remains off.
  • index.html becomes the default document, which avoids HTTP 403.14 at the site root.
  • The native endpoint still lives at iccIisIsapi.dll, so existing monitoring and smoke checks stay simple.
  • The browser tool console can upload ICC or XML input and run iccToXml, iccFromXml, iccDumpProfile, and iccRoundTrip on the server.

Quick tests

curl.exe http://localhost:18081/iccIisIsapi.dll?mode=health curl.exe http://localhost:18081/iccIisIsapi.dll curl.exe http://localhost:18081/iccIisIsapi.dll?format=xml Open http://localhost:18081/endpoints.html to upload an ICC or XML file.
