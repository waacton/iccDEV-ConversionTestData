Tool Workspaces

This root avoids HTTP 403.14 for /_tool-work/ and gives operators a stable navigation page. Individual job folders under this directory contain uploaded ICC or XML files, generated artifacts, and tool logs.

How to use it

  • Open Endpoint Console.
  • Upload an ICC profile or XML file.
  • Follow the returned workspace and artifact links from the browser results.
  • Use this root page any time you need a stable entry instead of IIS directory-level `403.14`.

Tool endpoint

The upload console runs iccToXml, iccFromXml, iccDumpProfile, and iccRoundTrip through POST /iccIisIsapi.dll?mode=tools. Each completed request leaves behind a per-request job directory under /_tool-work/.
